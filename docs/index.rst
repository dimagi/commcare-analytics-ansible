CommCare Analytics Ansible's Documentation
==========================================

Documentation for the `commcare-analytics-ansible <https://github.com/dimagi/commcare-analytics-ansible>`_
project, which contains tools for setting up and deploying `commcare-analytics <https://github.com/dimagi/commcare-analytics>`_.

We use the `Ansible <https://www.ansible.com/>`_ automation framework to provide one-click set up and management
of the commcare-analytics application a server.

This documentation site also includes a :doc:`guide for system administrators <system-administration>`
